## Ce que l'agent vocal fera

L'agent vocal est la version audio de ton agent SAV. Il décroche tes appels clients et leur parle — avec le même cerveau, le même ton, les mêmes connaissances que ton agent texte.

## Modalités prévues

- **Widget vocal** sur ton site Shopify (bouton "Appeler") — un appel WebRTC sans numéro à composer.
- **Numéro dédié** que tu peux mettre sur tes fiches produit, factures, signatures email.
- Voix synthétisée premium (ElevenLabs).
- Compatible avec ta config existante : ton, KB, guardrails, escalades.
